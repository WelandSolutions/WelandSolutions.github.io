Compact Store Flow REST API

API Key

To be able to make requests to the Compact Store Flow system an API key is required.
The API Key needs to be present in the "X-Api-Key" header of the request.

The API Key can be obtained from an administrator of the Compact Store Flow system.

Documentation

The documentation of the REST API can be found here

If you have access to a Compact Store Flow system, the REST API documentation can be found at http://[host]/integration.
When accessing this documentation there is also the possibility of making requests to the API using the "Try it out" feature.
