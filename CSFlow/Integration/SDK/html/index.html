Compact Store Flow SDK documentation.

Introduction

This is the documentation of the .Net SDK used for API-integration with the Compact Store Flow system.
Using the SDK makes it possible to perform operations in the Compact Store Flow system.

Operations

The SDK supports the following operations.

Orders

  • Create order
  • Add item to order
  • Review the outcome of order processing

Items

  • Create item
  • Update item
  • Review item balance

Getting started

The SDK is distributed as a NuGet package. To install the NuGet package do the following in Visual Studio:

  • From the menu, go to Tools/Options/NuGet Package Manager/Package Sources
  • Add a package source with
    Name: Weland Solutions
    Source: https://pkgs.dev.azure.com/WelandSolutions/079c9a5b-4524-4c10-af6d-caf8d1f6dbc8/_packaging/Compact_Store_Flow/nuget/v3/index.json
  • Add the NuGet package to a project by selecting "Weland Solutions" as package source in the "Manage NuGet packages" window
    and install the "CompactStoreFlow.SDK" package.

The starting point when using the SDK is Compact Store object, see CompactStore.

API Key

To use the SDK it needs to be configured with an API key, see CompactStoreConfig.
The API key can be obtained from an administrator of the Compact Store Flow system.

An API key is not needed if the SDK is configured for loopback mode.

Usage

For examples of SDK usage, see Compact Store Flow SDK usage examples.
These examples can be used as a starting point for implementing an integration towards Compact Store Flow.
